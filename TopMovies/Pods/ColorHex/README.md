## Require

- iOS 10+
- macOS 10.13+
- Xcode 9+

## Installation

### CocoaPods

```
pod "ColorHex"
```

## Usage

```
# red
UIColor.(hex:0xff0000)

# red
UIColor.(hex:0xff0000, alpha:0.1);
```
